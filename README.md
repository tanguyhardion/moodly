# Moodly

A daily mood tracking journal. Track your mood, energy, sleep, and focus with beautiful visualizations. All data stays private in your browser.

## Features

- Track mood, energy, sleep, and focus on a 1-5 scale
- Add daily notes
- View history and statistics
- Dark mode support
- 100% private - no servers, local storage only

## Getting Started

```bash
# Install dependencies
npm install

# Run dev server
npm run dev

# Build for production
npm run build
```

## Tech Stack

Built with Nuxt 4, Vue 3, and TypeScript.
